# Linux AI Assignment

This repository demonstrates AI-assisted troubleshooting and automation
for Linux virtual machines hosted on VMware vSphere.

## Coverage
- High CPU and memory diagnosis
- Automated recovery mechanisms

## Tools
- Linux utilities
- Bash scripting
- Python automation
- AI-assisted design

## Outcome
Improved reliability and reduced operational overhead.

## Assumptions
- VMware vSphere and VMware Tools are available
- Scripts are representative and can be extended for production use
- AI is used as an assistant, not a replacement for engineering judgment
